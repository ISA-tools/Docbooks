Future Developments

MIBBI checkout generated ISAconfigurations

ISA Team and MIBBI project are working together to enable creation of valid ISA configurations based on user input to create on the fly custom configurations. Those configurations would be maintained on a central server for all to use and would work towards enforcing Minimal Information annotation compliance

MIBBI checkout generated ISAconfigurations

The following screenshot shows the MICheckout portal where user input is used to create on the fly checklists which can be downloaded in various forms, which include ISA compatible configurations.

ISAconfiguration and RDF export

RDF representation of datasets is expanding. In order to allow for consistent creation of RDF data from ISAtools, the ISAconfigurator is being extended to formally specify the relations between entities and therefore allow mapping of ISA elements to existing ontological artifacts.
