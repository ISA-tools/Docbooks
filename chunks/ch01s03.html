Troubleshooting

I have Apache server running and port 8080 is already in use, How can I change this?

The browse view does not seem to work !

The most likely explanation is a mismatch between the path to index in the pom.xml used to build the web application and the loader hibernate.properties files. Please do check that both files rely on the same parameter and that read permissions are set correctly. This is essential for the database to resolve and access index.
