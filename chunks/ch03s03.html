Getting familiar with ISAconfigurator user interface

ISAconfigurator is a piece of software developed around the ISA-TAB format. The tool allows to define spreadsheet tables by using ISA-TAB syntactic elements. Therefore, ISAconfigurator have 2 main functions:

  1. Customize the default configuration to a community specific need.

    For instance, you may need to service users which mainly focus on Mouse experimentations, and therefore would like to ensure they use specific terminology and report information about diet, housing conditions, anatomy

  2. Define a new type of assay not covered in the default ISAconfiguration, while retaining the overall ISAsyntax

  3. Implement Minimal Information checklist
